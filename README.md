# Midipia v1: Minimalist 🎹🌈

A minimalist, color-locked MIDI visualizer where every note maps to a specific, vibrant hue.

## 🚀 How to Run
1. Clone this repository.
2. Open `index.html` in a MIDI-supported browser (Google Chrome or Microsoft Edge are recommended).
3. Connect your MIDI controller.
4. Play!

## ⚙️ Requirements
- **Browser:** Chrome, Edge, or any browser supporting the **Web MIDI API**.
- **Hardware:** Any MIDI-compatible keyboard or controller.
- **Connection:** USB or MIDI-to-USB interface.

## 🎨 Visual Logic
This version uses a **Pitch-to-Color** mapping. Each of the 12 notes in the chromatic scale is locked to a specific HSL hue, creating a consistent color identity for every note you play.
